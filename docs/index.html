Prerequisites.

Please make sure you have installed all prerequisites before you continue with this documentation:

  • Node.js
  • Grunt
  • Jekyll

Vendors.

Third party CSS/JavaScript libraries used in this theme:

  • Bootstrap
  • Font Awesome
  • jquery.hidenavbaronscroll.js
  • html5shiv.js
  • modernizr.js
  • respond.js
  • retina.js
  • skrollr.js

Installing Node.js Dependencies.

Open Terminal (Mac OS) or Command Prompt (Windows), access to the source folder and type:

$ npm install

In some cases, depending on your system configuration, you will need to run the task as administrator:

$ sudo npm install

This task will download and install all Node.js dependencies you need in order to run the jekyll site and store them at a folder named: node_modules Once the process ends, you need to run the following grunt command to publish and serve the jekyll site:

$ grunt watch

This process will concatenate all JavaScript files, compile LESS files into the main StyleSheet, copy all files to a folder at the same level of the source folder called: Kunstnerisk and start a server on port 1984 so you can access the site on your web browser with the following address:

http://localhost:1984/

Any change made to the source code (HTML/CSS/JS) while the process still running will fire up the compilation of the site and keep the server alive.

Notes.

A compiled version of the site is delivered with this theme, where the product is a simple HTML/CSS/JS site for any other use.

Configuring your site.

The site is fully customizable by editing the _config.yml file and rerunning the grunt process, all section of this one-page site are well documented.

Some examples of what you can edit

Main logo.

Just change the value fo both varialbes with the route of the regular and small logos

# Main Logo
# -------------------------

logo_regular: "media/logo/logo_regular.png"
logo_small_devices: "media/logo/logo_small.png"

Main Navigation Links

For each combination of title and url one item will be created on the menu, please consider that indentation and the dash (-) on the begining of each pair is required.

# Main Navigation Links
# -------------------------

navbar_links:
  - title: "Home"
    url: "#image-hero"
  - title: "Features"
    url: "#features"
  - title: "Services"
    url: "#services"
  - title: "Work"
    url: "#work"    
  - title: "Pricing Tables"
    url: "#pricing-tables"
  - title: "Contact"
    url: "#contact"

Navigation Background Diagonal Color

The bottom diagonal of the main navigation can be configured with four pre made colors.

# Navigation Background Color. 
# Options: caribbean-green, summer-sky, golden-poppy, deep-carmine-pink
navbar_color: "caribbean-green"
